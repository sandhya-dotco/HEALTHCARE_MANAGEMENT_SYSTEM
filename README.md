# Health-Care-Management-System

## Description
The Health Care Management System is a web-based application designed to manage and streamline healthcare services efficiently. It helps in maintaining patient records, managing appointments, and assisting healthcare staff in organizing daily operations digitally.

## Features
- User authentication (Admin/Doctor/Patient)
- Patient record management
- Appointment scheduling
- Doctor and staff management
- Stores previous history of patient
- Secure data storage
- Easy-to-use interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / Python  
- Database: MySQL  

## Purpose
This project aims to reduce manual work in healthcare centers, improve data accuracy, and provide quick access to medical information for better patient care.
